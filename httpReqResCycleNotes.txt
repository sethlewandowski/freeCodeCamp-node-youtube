Request Message (What USER sends)
- Method (get, post, put, delete)
- URL ()
- Headers (meta info about request) (key value pairs)
- Body (optional) (request payload)

Response Message (what SERVER sends)
- status text (OK)
- status code (200) (400 error) (401)
- Headers (content-type: application/json)
- Body
